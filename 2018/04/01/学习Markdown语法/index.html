强调

在强调内容两侧分别加上*或者_，如：

*斜体*，_斜体_
**粗体**，__粗体__
***斜体加粗体***，___斜体加粗体___

效果：

斜体，_斜体_

粗体，粗体

斜体加粗体，斜体加粗体

引用

文字引用

<blockquote><p>引用内容</p></blockquote>

引用可以嵌套。如果要在一个引用里插入一个引用，可以用两个>开头。以此类推。

> 这是一个引用。这是第一行
>
>这是第二行。
>> 这是一个嵌套的引用。这是第一行。
>>
>>这是第二行
>
> 外层引用的第三行。前面需要一个视觉上的空行表示内层嵌套的结束，空行前面的>可以有可以没有。

效果：

这是一个引用。这是第一行

这是第二行。

这是一个嵌套的引用。这是第一行。

这是第二行

外层引用的第三行。前面需要一个视觉上的空行表示内层嵌套的结束，空行前面的>可以有可以没有。

实例：

“焚诗书，坑术士” ——《史记》

代码引用

需要引用代码时，如果引用的语句只有一段，不分行，可以用 ` 将语句包起来。
如果引用的语句为多行，可以将```置于这段代码的首行和末行。

表格

示例：

| Tables        | Are           | Cool  |
| ------------- |:-------------:| -----:|
| col 3 is      | right-aligned | $1600 |
| col 2 is      | centered      |   $12 |
| zebra stripes | are neat      |    $1 |

效果：

col 2 is centered $12
